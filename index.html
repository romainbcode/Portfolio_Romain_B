Ce projet simule un site de paris sportif entièrement gratuit, uniquement fonctionnel avec de l'argent virtuel.
Le backend possède une API Rest developpé en python. Le frontend lui est développé avec Vuejs.
Deux bases de données ont été utilisé dans ce projet, une base de donnée non relationnelle : redis et une base de donnée relationnelle : SQLiteStudio.
Ce projet possède différentes pages tels qu'une interface de connexion et d'inscription, une page d'accueil où il est possible de voir le classement et les matchs d'une journée d'un championnat. Il est également possible de voir l'historique des différents paris effectués.

Ce projet est une copie simplifié de Twitter.
Le backend possède une API Rest developpé en python. Le frontend lui est développé avec vuejs.
Une seule base de données a été utilisé dans ce projet : redis, une base de donnée non relationnelle.
Ce twitter permet de twitter avec un nom d'utilisateur, de chercher des tweets avec un certains hashtag, de chercher les tweets d'une personne.
Deux dockerfiles sont disponibles dans le dossier du github afin de pouvoir lancer ce projet gràace à des conteneurs.
